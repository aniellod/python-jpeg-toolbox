# The JPEG Toolbox for Python

This project is in development and it is an unstable version. Please use it carefully.

## Install

```bash
pip3 install git+https://github.com/daniellerch/python-jpeg-toolbox --user
```

## Unnstall
```bash
pip3 uninstall jpeg-toolbox
```

```bash
python3
```

```bash
>>> img["coef_arrays"][0][0][0] = 85
```
